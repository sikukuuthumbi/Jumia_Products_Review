# Jumia Product Performance Dashboard (Excel)

## Project Overview

This project analyzes a Jumia product dataset using Microsoft Excel to uncover insights about product pricing, discounts, customer reviews, and ratings. The objective was to clean the dataset, perform exploratory analysis, and build an interactive dashboard for decision-making.

## Dataset

The dataset contains information about Jumia products including:

* Product Name
* Current Price
* Old Price
* Discount Percentage
* Number of Reviews
* Product Rating

## Tasks Completed

### 1. Data Cleaning

The raw dataset was cleaned by:

* Removing duplicate records
* Converting price columns from text to numeric values
* Standardizing rating values into numeric format
* Correcting invalid review values
* Preparing the dataset for analysis

### 2. Data Enrichment

Additional analytical columns were created:

* **Absolute Discount** = Old Price − Current Price
* **Discount Category**

  * Low Discount
  * Medium Discount
  * High Discount
* **Rating Category**

  * Poor
  * Average
  * Excellent

### 3. Data Analysis

The following analyses were performed:

* Average Current Price
* Average Old Price
* Average Discount Percentage
* Average Product Rating
* Top 5 Highest Rated Products
* Products with Lowest Ratings
* Top 10 Products with Highest Discounts
* Top 10 Products with Most Reviews
* Relationship between Discounts and Reviews
* Relationship between Ratings and Reviews

### 4. Dashboard Development

An interactive Excel dashboard was created using:

* Pivot Tables
* Pivot Charts
* Slicers
* Conditional Formatting

Dashboard sections include:

* Product Performance
* Product Categories
* Discount vs Reviews Analysis
* Rating vs Reviews Analysis
* Top Product Rankings
